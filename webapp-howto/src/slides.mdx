# First Challenge

<OrderedList>
    <ListItem>{'Bootstrap environment using a custom codespace and template repository'}</ListItem>
    <ListItem>{'Run tests'}</ListItem>
    <ListItem>{'Use the simple API that returns a list of pizzas at /api/pizzas'}</ListItem>
    <ListItem>{'render() the list'}</ListItem>
    <ListItem>{'Run tests'}</ListItem>
</OrderedList>

---

# Build It

* [`useEffect`](https://beta.reactjs.org/reference/react/useEffect) to fetch data
* [`useState`](https://beta.reactjs.org/reference/react/useState) to store an in-memory copy of the `/api/pizzas` response and cause page to re-render

remember you can use `npm test` for fast feedback

---

# Build It

* more [`useState`](https://beta.reactjs.org/reference/react/useState)
* [`useCallback`](https://beta.reactjs.org/reference/react/useCallback) to bind an event handler for search input

remember you can use `npm test` for fast feedback

---

# Third Challenge

* use a [`next/link`](https://nextjs.org/docs/api-reference/next/link) to navigate to details of a pizza
* add a [dynamic route](https://nextjs.org/docs/basic-features/pages#pages-with-dynamic-routes) to show the details
* add [additional api](https://nextjs.org/docs/api-routes/dynamic-api-routes) at `/api/pizzas/id` to get details of specific pizza

---

# Build It

* add a folder in `pages/api` called `pizzas`
* add a file named `[id].js` in `pages/api/pizzas` and implement API
* add a folder in `pages` called `pizzas`
* add a file named `[id].js` in `pages/pizzas` and implement showing the details of one pizza
* add a link in `index.js`
